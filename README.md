# ClignosisUI

## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000/`  

## Pages

#### Login
`localhost:3000/`  

#### Home
`localhost:3000/home`  

#### Cards
`localhost:3000/cards`  

#### Regulatory 360
`localhost:3000/timeline`
